# Léo - Mercadinho Simples

Sistema básico de mercadinho feito em Python com FastAPI.

**Funcionalidades:**
- Login admin (username: admin / senha: admin123)
- Login usuário comum
- Cadastro/edição/exclusão de até 100 produtos
- Listagem de produtos pro cliente
- Carrinho simples + pagamento Pix (QR Code) ou dinheiro

**Instalação**  
```bash
pip install -r requirements.txt
python app.py
